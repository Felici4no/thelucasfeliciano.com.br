Manifesto

Sobre o projeto

Este espaço nasceu da convicção de que som e forma são inseparáveis. Cada faixa é um documento afetivo — uma decisão estética comprometida com honestidade e precisão.

Não busco volume, mas profundidade. Não produto, mas presença. O trabalho existe para quem está disposto a ouvir com atenção o que muitas vezes não se diz em palavras.

O exercício é contínuo. Cada lançamento é um registro de onde estou — não onde quero chegar. A obra não é um destino, é um diário.

Princípios

  • —
    Foco

    Menos, porém melhor. Cada escolha é uma renúncia consciente.

  • —
    Craft

    Compromisso com o processo tanto quanto com o resultado.

  • —
    Constância

    Presença regular, sem urgência. O tempo é parte da obra.
